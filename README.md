# About
前端学习实验楼,基于`JavaScript`实现多个不同的形状图来组成一个逼真的玫瑰花。共使用了 31 个形状：24 个花瓣，4 个萼片，2 个叶子和 1 根花茎，其中每一个形状图都用代码进行描绘。我们可以学到 JavaScript 及计算机图形的一些知识。<br>
